## Detailed Business Rules

### 1. Office Hours & Time Slots
- **Clinic hours**: 8:00 AM – 4:00 PM
- **Blocked times**:
  - 12:00 PM – 1:00 PM (Lunch)
  - 4:00 PM – 5:00 PM (Paperwork)
- **Available slots**: 8AM, 9AM, 10AM, 11AM, 1PM, 2PM, 3PM (7 slots per day)

### 2. Daily Slot Management (Capacity Units)
- **Total daily capacity**: 8 units
- **Service weights**:
  - Kapon = 2 units
  - Vaccination = 1 unit
  - Adoption Interview = 1 unit

### 3. Merge Logic
- If a user books a second service for the **same pet on the same day**, append to existing appointment
- One pet + multiple services = one time slot but cumulative weight
- Merged appointment stores `merged_pet_ids[]` in metadata

### 4. Adoption Interviews
- Support **Both** options: Zoom or In-Person
- If 'Zoom' selected: include `meeting_link` in metadata JSON

## Database Transaction Logic (Prevent Double-Booking)

```php
DB::transaction(function () use ($data) {
    // 1. Lock existing appointments for this date/time
    $existing = Appointment::where('appointment_date', $date)
        ->where('appointment_time', $time)
        ->lockForUpdate()
        ->first();

    // 2. Calculate new weight
    $newWeight = $this->calculateWeight($serviceType, $petIds);

    // 3. Check capacity (8 - existing_weight >= new_weight)
    if ($existing && (8 - $existing->total_weight < $newWeight)) {
        throw new \Exception('Capacity exceeded');
    }

    // 4. Check merge logic (same pet, same day)
    $mergeAppointment = $this->findMergeCandidate($date, $petIds);
    if ($mergeAppointment) {
        // Append to existing - update weight and metadata
        $mergeAppointment->total_weight += $newWeight;
        $mergeAppointment->save();
        return $mergeAppointment;
    }

    // 5. Create new or update existing
    if ($existing) {
        $existing->total_weight += $newWeight;
        $existing->save();
        return $existing;
    }

    return Appointment::create([...]);
});
```
